🎓 Classroom Availability System

A web-based system designed to efficiently manage classroom bookings, avoid scheduling conflicts, and streamline academic timetable management.

📌 Features
🏫 Manage classrooms, departments, and academic years
👨‍🏫 Faculty and subject management
📅 Timetable slot creation and management
🚫 Prevent double-booking of rooms and faculty
🔐 Basic authentication system
🧱 Project Structure

⚙️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
